2. GitHub Desktop - Aplicativo - https://desktop.github.com/

Nesse app posso fazer as mesmas coisas que no github normal, porém ele é mais direcionado para criação e clonagem de repositorio que o github em si

Posso adicionar um diretório existente no meu pc

Para conseguir realizar as operações no git desktop, devo me logar com a conta do github: file -> option -> accounts -> logar

Depois disso, antes de fazer algumas operações, eu tenho que fazer o fetch, que é sincronizar as operações do git desktop com as do github -> clicando no canto superior -> fetch origin

Após dar um commit, eu posso fazer o push origin, que fica no mesmo local do fetch origin
