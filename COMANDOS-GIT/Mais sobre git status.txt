No git status aparecem algumas informações:

HEAD - Estado atual do codigo, onde o Git os colocou

Working Tree - Local onde os arquivo realmente estão armazenados e editados

index - Local onde o git armazena o que será commitado, local entre a working tree e o repositorio Git em si
