ARTIGO ALURA

Como trabalhar com essa linguagem de markup?

Markdown é um formato simples de markup(marcação de texto)


Deixando texto em negrito:
**textoEmNegrito**
Ex de uso:
Meu nome é **Luca**.

Coloando um link dentro de um texto:
[O link vai ser guardado dentro dessa frase](https://linkExemplar.com)
Ex de uso:
Aqui tem uma [página com os códigos mais frequentes](https://link) do Markdown
