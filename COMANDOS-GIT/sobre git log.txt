git log --oneline

Esse comando mostra o log de maneira resumida



git log -p
Esse comando mostra mais informações sobre os logs



git log --author="user_name"
Pra pesquisar informações do autor daquele commit



git log --since=1.month.ago --until=1.day.ago
Pra pesquisar informações por data




git log --pretty="format:%h %s"
Formatando a visão das informações



Mais formas de exibir informações do commit - https://devhints.io/git-log-format
Mais parametro do git log - https://devhints.io/git-log
